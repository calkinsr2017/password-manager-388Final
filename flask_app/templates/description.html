Our website is a password management system (i.e. lastpass) named The Vault. Users will be able to create and access an account where they can store usernames and passwords for specific apps or websites. There will be other functionalities such as storing the url of the website. Users will be able to organize and navigate the system by searching for websites by name or selecting the view all stored passwords buttons.
